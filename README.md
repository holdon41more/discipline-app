> - **iOS 用户**：使用 Safari 打开，点击“分享”->“添加到主屏幕”。
> - **Android 用户**：使用 浏览器 打开，点击“添加到桌面”。

## ✨ Workshop 专项升级
- **AI 陪伴系统**：基于 **Alibaba Qwen API**，通过 **Prompt Engineering** 实现情感化激励反馈（彩虹屁）。
- **极速协同开发**：全程使用 **Trae (AI-Native IDE)** 进行自然语言驱动的功能重构与逻辑解耦。
- **PWA 性能优化**：完善了 Service Worker 逻辑，支持一键添加至手机主屏幕，提升自律养成仪式感。
